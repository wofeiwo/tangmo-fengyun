## 一、时代背景

### 1.1 时间锚点

**核心纪元**: 唐僖宗乾符二年（875年）至后周显德七年（960年）

**关键时间节点**:
| 年份 | 事件 | 影响 |
|------|------|------|
| 875年 | 王仙芝起义 | 唐末民变开端 |
| 878年 | 黄巢起义 | 动摇唐朝根基 |
| 880年 | 黄巢攻占长安 | 唐僖宗出逃 |
| 884年 | 黄巢败亡 | 藩镇割据加剧 |
| 907年 | 朱温篡唐，建立后梁 | 唐朝灭亡，五代开始 |
| 923年 | 李存勖灭梁，建立后唐 | 五代更替 |
| 936年 | 石敬瑭割让燕云十六州 | 契丹南下隐患 |
| 960年 | 赵匡胤陈桥兵变 | 宋朝建立 |

### 1.2 世界状态

**政治格局**: 
- 唐朝中央权威崩塌，藩镇割据
- 907年后进入"五代十国"时期
- 53年间中原更替5个王朝（后梁、后唐、后晋、后汉、后周）
- 南方同时存在10个割据政权（前蜀、后蜀、南吴、南唐、吴越、闽、楚、南汉、南平、北汉）

**社会状态**:
- 战乱频繁，民不聊生
- 士族门阀衰落，寒门崛起
- 人口锐减，经济凋敝
- 科举制度时断时续

**军事状态**:
- 藩镇拥有独立军队（牙兵）
- 中央禁军衰微
- 骑兵地位上升（沙陀、契丹影响）

### 2.2 势力演变图

```
875-884年（黄巢之乱）
├── 唐朝中央（衰微）
├── 起义军：王仙芝→黄巢
└── 勤王藩镇：朱温、李克用、时溥等

884-907年（唐末割据）
├── 朱温（宣武）: 吞并河南诸镇
├── 李克用（河东）: 坚守山西
├── 杨行密（淮南）: 割据江淮
└── 其他藩镇：各自为政

907-960年（五代十国）
├── 五代（中原）: 梁→唐→晋→汉→周
├── 十国（南方）: 各自独立
└── 外部威胁：契丹（辽）崛起
```

### 2.3 核心势力详解

#### 朱温（后梁太祖）
- **出身**: 贫寒，黄巢部将，后降唐
- **特点**: 狡诈多疑，善用权谋
- **优势**: 占据中原腹地，人口众多
- **劣势**: 名声不佳，士族不附

#### 李克用（后唐奠基者）
- **出身**: 沙陀族，世袭河东节度使
- **特点**: 勇猛善战，重义气
- **优势**: 精锐骑兵（鸦军）
- **劣势**: 民族身份受歧视

#### 杨行密（南吴奠基者）
- **出身**: 庐州豪强
- **特点**: 宽厚待民，善用人
- **优势**: 江淮富庶，水军强大
- **劣势**: 北方骑兵威胁

## 三、官制体系

### 3.1 中央官制（唐制延续）

**三省六部**:
| 机构 | 职能 | 主要官职 |
|------|------|----------|
| 中书省 | 草拟诏令 | 中书令、中书侍郎 |
| 门下省 | 审核诏令 | 侍中、门下侍郎 |
| 尚书省 | 执行政令 | 尚书令、左右仆射 |
| 吏部 | 官员任免 | 尚书、侍郎 |
| 户部 | 户籍财政 | 尚书、侍郎 |
| 礼部 | 礼仪科举 | 尚书、侍郎 |
| 兵部 | 军事行政 | 尚书、侍郎 |
| 刑部 | 司法刑狱 | 尚书、侍郎 |
| 工部 | 工程营造 | 尚书、侍郎 |

**使职差遣**（实际权力）:
- 枢密使：军事决策（晚唐兴起）
- 盐铁使：财政税收
- 度支使：财政支出
- 观察使：地方监察

### 3.2 地方官制

**藩镇体系**:
| 职位 | 品级 | 职能 | 备注 |
|------|------|------|------|
| 节度使 | 从二品 | 军政合一，辖区最高长官 | 实际割据 |
| 观察使 | 正三品 | 监察州县 | 常由节度使兼任 |
| 刺史 | 正四品 | 州级行政长官 | 受节度使节制 |
| 县令 | 正六品 | 县级行政长官 | 基层实权 |

**节度使权力**:
- 军事：统率牙兵，自行招募
- 财政：截留赋税，自铸钱币
- 人事：自行任命僚属
- 司法：境内案件终审

## 四、军事制度

### 4.1 军队构成

| 类型 | 来源 | 特点 | 忠诚度 |
|------|------|------|--------|
| 牙兵 | 节度使亲兵 | 精锐，待遇优厚 | 高（但易哗变） |
| 州兵 | 各州征调 | 战斗力一般 | 中 |
| 乡兵 | 民间招募 | 农忙务农，农闲训练 | 低 |
| 蕃兵 | 少数民族 | 骑兵精锐 | 中（以利维系） |
| 禁军 | 中央直属 | 唐末衰微 | 低（常被藩镇吞并） |

### 4.2 典型军队

**朱温宣武军**:
- 核心：牙兵约1万
- 特点：步骑混合，纪律严明
- 战术：善用伏击，夜袭

**李克用河东军**:
- 核心：鸦军（沙陀骑兵）约5000
- 特点：重装骑兵，冲击力极强
- 战术：正面冲锋，侧翼包抄

**杨行密淮南军**:
- 核心：水军约2万
- 特点：战船众多，水战专家
- 战术：水陆配合，封锁河道

### 4.3 武器装备

**步兵装备**:
- 主武器：陌刀、长枪、弓弩
- 防具：明光铠、皮甲
- 辅助：盾牌、绊马索

**骑兵装备**:
- 主武器：马槊、弯刀、角弓
- 防具：铁甲、马铠
- 辅助：套索、飞爪

**攻城器械**:
- 抛石机、攻城锤、云梯
- 火攻：火油、火箭（早期火药未成熟）

## 五、社会结构

### 5.1 阶层划分

| 阶层 | 构成 | 地位 | 流动性 |
|------|------|------|--------|
| 皇室/宗室 | 李唐宗族 | 名义最高，实际衰微 | 无 |
| 士族门阀 | 崔、卢、李、郑等 | 传统高贵，逐渐衰落 | 低 |
| 寒门官僚 | 科举出身 | 新兴势力，崛起中 | 中 |
| 武将藩镇 | 节度使及僚属 | 实际掌权者 | 高（乱世机遇） |
| 自耕农 | 有地农民 | 纳税主体 | 低 |
| 佃农 | 无地农民 | 依附地主 | 低 |
| 商人 | 行商坐贾 | 地位低下，但富裕 | 中 |
| 奴婢 | 家奴、官奴 | 最低阶层 | 极低 |

### 5.2 士族衰落与寒门崛起

**士族代表**:
- 清河崔氏、范阳卢氏、陇西李氏、荥阳郑氏
- 特点：累世为官，门生故旧遍布朝野
- 衰落原因：战乱摧毁庄园经济，科举冲击

**寒门代表**:
- 朱温（贫农出身）、钱镠（私盐贩子）、王建（屠夫）
- 特点：凭军功或机遇崛起
- 崛起原因：乱世需要人才，门第观念淡化

### 5.3 科举制度

**唐末状态**:
- 时断时续（战乱影响）
- 录取人数锐减（每年约20-30人）
- 考试内容：诗赋、经义、策论

**五延续**:
- 后唐恢复科举
- 后周完善制度
- 为宋朝科举盛世奠基

## 六、经济与文化

### 6.1 经济状况

**农业**:
- 北方：战乱破坏，人口南迁，田地荒芜
- 南方：相对稳定，水利兴修，稻作发展
- 赋税：两税法延续，但藩镇截留严重

**商业**:
- 北方：丝路中断，贸易萎缩
- 南方：海上贸易兴起（广州、泉州）
- 货币：铜钱短缺，铅铁钱、布帛并行

**手工业**:
- 陶瓷：越窑青瓷、邢窑白瓷
- 纺织：吴越蜀锦、江南丝绸
- 冶炼：铁器、铜镜

### 6.2 文化状态

**文学**:
- 诗歌：晚唐余韵（韩偓、韦庄）
- 词：花间词派兴起（温庭筠、韦庄）
- 散文：古文运动延续

**宗教**:
- 佛教：禅宗兴盛，寺院经济膨胀
- 道教：皇室崇奉，但影响力下降
- 民间信仰：鬼神崇拜盛行

**科技**:
- 雕版印刷：逐渐普及
- 火药：初期军事应用
- 航海：指南针雏形

## 八、穿越者设定建议

### 8.1 身份选择

**推荐身份**:
| 身份 | 优势 | 劣势 |
|------|------|------|
| 寒门士子 | 可科举入仕，上升空间大 | 无背景，需自力更生 |
| 藩镇幕僚 | 接近权力核心，影响力大 | 卷入政治斗争风险高 |
| 商人 | 经济自由，信息灵通 | 社会地位低 |
| 乡绅 | 地方影响力，根基稳固 | 易成军阀目标 |

**不推荐身份**:
- 皇室宗室（易成政治牺牲品）
- 高官显贵（乱世首当其冲）
- 平民百姓（生存压力大）

### 8.2 金手指限制

**原则**: 少金手指，写实为主

**可接受设定**:
- 现代知识（历史走向、科技原理）
- 记忆力超群（背诵经典、地图）
- 体能略优于常人（适度锻炼）

**不可接受设定**:
- 系统/面板
- 超自然能力
- 现代武器直接出现

### 8.3 信息差优势

**历史预知**:
- 知道重大事件时间节点
- 知道各势力最终结局
- 知道关键人物性格特点

**技术优势**:
- 基础科学原理（物理、化学）
- 农业改良（选种、肥料）
- 医疗常识（消毒、隔离）
