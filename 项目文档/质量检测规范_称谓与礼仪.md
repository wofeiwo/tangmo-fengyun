## 🎯 核心原则

> **中古时代阶级分明，称谓必须反映身份差距和社交关系。**

**违者处理**：
- 写作中发现 → 立即修正
- 审核中发现 → 扣分（细节度 -5 分/处）
- 发布后发现 → 修订版本

## 一、称谓体系（按身份）

### 1.1 平民对官员/军官

| 场景 | 正确称谓 | 错误称谓 | 示例 |
|------|---------|---------|------|
| 自称 | 草民/小人/小的 | 我/某家 | "草民李三郎" |
| 称对方（知职位） | 大人/老爷/官人 | 直呼其名 | "参见大人" |
| 称对方（知军职） | 都头/校尉/将军 | 王都头/刘校尉 | "都头饶命" |
| 称对方（不知职位） | 官爷/军爷 | 那个当兵的 | "官爷行个方便" |

**常见错误**：
```
❌ "王都头，草民有事禀报。"
✅ "都头，草民有事禀报。"

❌ "刘校尉，求您放过我们。"
✅ "大人，求您放过我们。"
```

---

### 1.2 平民对平民（陌生）

| 场景 | 正确称谓 | 错误称谓 | 示例 |
|------|---------|---------|------|
| 称长辈（男） | 老丈/翁/阿公 | 老头/老王 | "这位老丈" |
| 称长辈（女） | 婆婆/阿婆 | 老太婆 | "婆婆行个方便" |
| 称平辈（男） | 郎君/大哥/阿兄 | 兄弟/哥们 | "这位郎君" |
| 称平辈（女） | 娘子/大嫂/阿姐 | 大姐 | "这位娘子" |
| 称晚辈 | 小郎君/小娘子 | 小孩 | "小郎君从哪里来" |

**常见错误**：
```
❌ "王七丈，你去哪？"
✅ "王老丈，你去哪？"

❌ "老头，你知道路吗？"
✅ "老丈，请问此去汴州怎么走？"

❌ "兄弟，借个火。"
✅ "郎君，借个火。"
```

---

### 1.3 平民对平民（熟悉）

| 关系 | 正确称谓 | 示例 |
|------|---------|------|
| 同辈朋友 | 姓 + 排行 + 郎（王七郎） | "七郎今日有空？" |
| 长辈 | 姓 + 老 + 排行（王老七） | "王老丈说得是" |
| 晚辈 | 小名/排行 | "三郎过来" |

---

### 1.4 军官对下属

| 场景 | 正确称谓 | 错误称谓 | 示例 |
|------|---------|---------|------|
| 称呼士兵 | 尔等/你等/这厮 | 兄弟们/大伙 | "尔等听令！" |
| 称呼亲兵 | 直接叫名字/排行 | 小王小李 | "李三郎，过来" |
| 自称 | 我/本官/本将 | 某家/老子 | "本将在此" |

**常见错误**：
```
❌ "兄弟们，跟我冲！"
✅ "尔等，随我冲锋！"

❌ "小王，去打探消息。"
✅ "李三郎，去打探消息。"
```

---

### 1.5 军官对军官（同级）

| 场景 | 正确称谓 | 示例 |
|------|---------|------|
| 正式场合 | 姓 + 官职 | "刘校尉请" |
| 非正式场合 | 姓 + 排行 + 郎 | "王七郎好久不见" |
| 自称 | 我/小弟 | "小弟有一事相求" |

---

### 1.6 军官对军官（上级）

| 场景 | 正确称谓 | 示例 |
|------|---------|------|
| 自称 | 末将/卑职/属下 | "末将遵命" |
| 称对方 | 大人/将军/都头 | "都头有何吩咐" |

## 二、社交礼仪细节

### 2.1 见面礼

| 身份差距 | 礼仪 | 示例 |
|---------|------|------|
| 平民见官员 | 下跪/作揖 | "草民参见大人" |
| 下属见上级 | 拱手/作揖 | "末将参见都头" |
| 平辈相见 | 拱手/抱拳 | "李郎君别来无恙" |
| 长辈见晚辈 | 点头即可 | "三郎来了" |

### 2.2 对话语气

| 身份差距 | 语气 | 示例 |
|---------|------|------|
| 对上级 | 恭敬/谦卑 | "大人明鉴" |
| 对平辈 | 平等/客气 | "郎君请" |
| 对下级 | 直接/命令 | "你去办这件事" |
| 对平民 | 居高临下 | "你这刁民" |

### 2.3 自称变化

| 身份 | 自称 | 使用场景 |
|------|------|---------|
| 平民对官 | 草民/小人 | 正式场合 |
| 平民对官 | 我 | 非正式场合 |
| 军人对上级 | 末将/卑职 | 正式场合 |
| 军人对同级 | 我/小弟 | 非正式场合 |
| 读书人 | 小生/学生 | 对长辈/老师 |
| 年长者 | 老朽/某家 | 对晚辈 |

## 三、黄巢起义特定称谓

### 3.1 不同立场的称谓

| 立场 | 称谓黄巢 | 称谓起义军 |
|------|---------|-----------|
| 朝廷官员 | 黄巢逆贼/反贼 | 叛军/贼寇 |
| 平民（中立） | 黄王/黄帅 | 义军/大军 |
| 起义军内部 | 主公/大王 | 我军/义师 |
| 主角（穿越者） | 黄巢/黄王 | 义军/起义军 |

**常见错误**：
```
❌ 平民直接说："黄巢打过来了。"
✅ 平民说："黄王打过来了。" / "义军打过来了。"

❌ 官员说："黄王率军前来。"
✅ 官员说："黄巢逆贼率军前来。"
```

## 四、姓名格式规范

### 4.1 平民姓名

| 类型 | 格式 | 示例 |
|------|------|------|
| 正式姓名 | 姓 + 名（字） | 李守业（字定国） |
| 日常称呼 | 姓 + 排行 + 郎 | 李三郎 / 王七郎 |
| 长辈称呼 | 姓 + 老 + 排行 | 王老七 / 李老四 |
| 自称 | 草民 + 姓名 | "草民李三郎" |

### 4.2 军官姓名

| 类型 | 格式 | 示例 |
|------|------|------|
| 正式介绍 | 番号 + 官职 + 姓 + 名 | "宣武军都头王彪" |
| 日常称呼 | 姓 + 官职 | 王都头 / 刘校尉 |
| 自称 | 官职 + 名/我 | "本都头" / "我" |

## 五、质量检测清单

### 5.1 写作自检（刘一手）

每章写完必查：

- [ ] 平民对官员是否用敬称（大人/都头）？
- [ ] 下属对上级是否用谦称（末将/草民）？
- [ ] 长辈称谓是否正确（老丈/翁，非"某丈"）？
- [ ] 平辈称谓是否有"郎君/郎"？
- [ ] 军官对士兵是否体现权威（尔等/你等）？
- [ ] 自称是否符合身份（草民/末将/我）？
- [ ] 黄巢称谓是否符合说话人立场？
- [ ] 对话语气是否符合身份差距？

### 5.2 审核检查（主编）

审核时重点关注：

| 检查项 | 扣分标准 | 示例 |
|--------|---------|------|
| 称谓错误 | -5 分/处 | "王七丈"→"王老丈" |
| 自称错误 | -5 分/处 | "我"→"草民"（平民对官） |
| 语气不符 | -3 分/处 | 军官对士兵太亲切 |
| 黄巢称谓错误 | -5 分/处 | 平民直呼"黄巢" |

**通过标准**：称谓错误 ≤2 处/章

## 六、场景示例库

### 6.1 军营场景

```
【正确示例】
校尉："你，叫什么名字？"
士兵："回校尉，草民李三郎。"
校尉："多大了？"
士兵："二十有二。"
校尉："读过书？"
士兵："识得几个字。"
```

### 6.2 流民营场景

```
【正确示例】
A："这位老丈，从何处来？"
B："洛阳城。小郎君呢？"
A："白马寺附近。"
B："那也不远了。"
```

### 6.3 军官之间场景

```
【正确示例】
A："刘校尉，今日有空？"
B："王都头请。有何指教？"
A："小事一桩，借一步说话。"
```

### 6.4 错误修正示例

```
【原文】
"王七丈，你知道这铜钱是哪来的吗？"
"黄巢的兵来了，快跑！"
"兄弟们，跟我冲！"

【修正后】
"王老丈，你知道这铜钱是哪来的吗？"
"黄王的兵来了，快跑！"
"尔等，随我冲锋！"
```
