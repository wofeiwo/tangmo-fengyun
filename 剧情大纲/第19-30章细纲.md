# 第24章细纲：《成长》【优化版】

> **审核反馈**：增加挫折情节，提升冲突度

## 核心：主角在朱温麾下快速成长，但遭遇挫折

### 5幕结构
1. **屡立战功** - 主角连续几场战役获胜，晋升
2. **突发危机** - 有人告密主角私通敌军（⚠️新增冲突）
3. **调查期间** - 主角被暂时扣押，但最终澄清
4. **虽胜犹损** - 战功被削减，但获得教训
5. **卷末钩子** - 朱温称王，但主角开始考虑退路

### 优化点
- 增加"告密"情节 → 冲突度↑
- 增加"被扣押"情节 → 悬念度↑
- 主角学会隐藏实力

# 第28章细纲：《养兵》【优化版】

> **审核反馈**：压缩节奏，加快推进

### 5幕结构（压缩版）
1. **战后快速整编** - 简短交代战后状况，直接进入正题
2. **招兵与练兵** - 并行进行，突出效率（⚠️压缩）
3. **根据地选址** - 徐先生献策，主角快速决断
4. **势力初成** - 简洁展示经营成果
5. **卷末钩子** - 朱温猜忌上门（加快节奏）

### 优化点
- 删减冗余场景描述
- 每幕聚焦一个核心动作
- 节奏加快，冲突前置
