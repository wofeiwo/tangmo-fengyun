# 第一卷大纲：《乱世求生》

> **参考文档**：
> - 世界观：`唐末风云_世界观白皮书_v1.1.md`
> - 角色：`01-主角团人设方案.md`（主角：**李守业**，字定国）
> - 历史人物：`02-历史人物设计.md`

**卷名**：乱世求生
**章节**：15章
**字数**：约10万字
**核心**：穿越→生存→军事立足
**评分目标**：90/100分

---

## 一、卷核心设计

### 1.1 主线
```
穿越→流民身份→求生→参军→小胜→站稳脚跟
```

### 1.2 节奏曲线
```
第1-3章：穿越 + 身份确认 + 生存危机（紧张）
第4-6章：求生挣扎 + 小机遇（缓中有紧）
第7-9章：参军 + 初战（上升）
第10-12章：战功 + 内部矛盾（高潮）
第13-15章：站稳 + 新危机伏笔（回落 + 悬念）
```

### 1.3 情绪设计
| 章节 | 情绪 | 目的 |
|------|------|------|
| 1-3 | 紧张/迷茫 | 建立代入感 |
| 4-6 | 压抑/希望 | 积累情绪 |
| 7-9 | 振奋 | 首次释放 |
| 10-12 | 紧张→释放 | 高潮 |
| 13-15 | 满足 + 悬念 | 卷末收束 |

## 二、章节详细设计

### 第1章《醒来乱世》
**核心情节点**：
- 穿越醒来，发现自己身处唐末战乱（约878年，黄巢起义期间）
- 身份：流民，与家人失散
- 环境：逃难路上，周围都是饥民
- 金手指：仅有现代知识（历史走向、基础军事/科技常识）

**冲突设计**：
- 生存冲突：饥饿、疲惫、随时可能死亡
- 认知冲突：现代人思维 vs 乱世现实

**悬念/钩子**：
- 开篇：我是谁？这是哪里？
- 中段：乱兵来袭，逃难人群四散
- 结尾：主角决定不再被动逃难

**历史背景**：黄巢起义军与唐军交战，流民四散

**评分支撑**：
- 节奏感：快速切入乱世危机
- 冲突度：生存压力
- 悬念度：身份 + 环境悬念
- 创新性：弱金手指设定
- 细节度：逃难场景细节

---

### 第2章《流民》
**核心情节点**：
- 融入流民群体，了解当前局势
- 得知黄巢起义军逼近，官府抓壮丁
- 遇到第一个重要配角：老流民张伯（经验丰富的老兵）
- 张伯讲述当前局势（信息传递）

**冲突设计**：
- 社会冲突：流民内部资源争夺
- 认知冲突：主角现代思维 vs 古人思维

**悬念/钩子**：
- 开篇：如何活下去？
- 中段：张伯身份疑点（为何懂军事？）
- 结尾：官府来抓壮丁

**历史背景**：唐末抓壮丁常态

**评分支撑**：
- 节奏感：信息展示 + 危机来临
- 冲突度：资源争夺
- 悬念度：张伯身份
- 细节度：流民生活细节

**冲突设计**：

**悬念/钩子**：
- 开篇：能逃吗？

- 结尾：主角决定在军队中找机会

**历史背景**：唐末地方团练制度

**评分支撑**：
